Running the Liftpass Dashboard

The easier way to interact with Liftpass API and manage/configure application content is using the dashboard. The Dashboard can be built using the build.py script which also genereates the documentation for Liftpass. The dashboard will be generated and placed into the folder liftpass/site/build/dashboard/

{% highlight 'bash' %} cd site python build.py {% endhighlight %}

To run the dashboard the easiest way is to start Python's native HTTP server from within the built site document folder:

{% highlight 'bash' %} cd build python -m http.server {% endhighlight %}

From your browser you can then navigate to the URL http://127.0.0.1:8000/dashboard/.
