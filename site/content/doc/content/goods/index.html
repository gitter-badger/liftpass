Goods

There are two kinds of goods Liftpass manages:

Virtual Goods are those that the user buys using virtual currency.

In-App Purchases are those that are purchased by the user using real money.

Usage

Goods must be registered with Liftpass before they can be used in the SDK. The name of the virtual goods should reflect the names as they appear in the code of the application.

Additionally the name of the goods are also used in the pricing sets entered in Liftpass. This is because in all pricing engines the final price the user sees for a good is retrieved from a value-key dictionary where the keys are the names of the goods as they appear in the code.
