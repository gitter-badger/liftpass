Content Management

The core of Liftpass is the content management of the applications currencies, progress metrics, goods, and prices. When creating an application in Liftpass each of these sections must be populed to reflect how the application operates.

Currencies
Currencies are the ways the player can buy and spend inside the game with virtual currency.

Metrics
Liftpass records up to 32 different progress metrics. These are user defined and should reflect the game features and developer's interests.

Goods
Goods represent the virtual items the player can purchase within the application. Additionally goods are also IAP that can be purchased for real money.

Prices
Prices is where Currencies, Metrics and Goods come together so that every player can have a unique set of prices.

Managing the content

There are primary two ways to manage the content of the application:

  1. A dashboard is included with the Liftpass server and can be accessed via any modern browser. The address is: http://server-name.com/dashboard/.
  2. Alternatively, there is a full feature API that allows for managing every single aspect of Liftpass. More information can be found in the API page.
