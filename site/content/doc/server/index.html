Liftpass Server

Liftpass is broken down in to 3 principle parts:

API server manages all API requests which includes the calls made by the SDK for updating the application with new prices and transmitting analytics information.

Analytics processing which processes the analytics information that was sent from the SDK.

The Dashboard Along with the Liftpass website and documentation that is built as part of the server, the dashboard enables developers to quickly manage application content via the dashboard.

These three components are designed to be modular and easily scalable and reconfigured to fit the size applications that it will serve.
