Get Started

You should be able to build Liftpass on most Linux-based systems.

Get the build dependencies

Liftpass is developed in Python 3.x. There are a number of modules required for running the server, however all are available through pip and easy_install.

  • Flask
  • SQLAlchemy
  • Requests

The setup.py script can install these dependencies on most modern systems that have Python 3.x and SetupTools installed.

Get the source code

Download or clone the Liftpass repository: {% highlight 'bash' %} # Clone the repository git clone https://github.com/whiteshoemedia/gondola-core.git # Download repository as a zip file wget https://github.com/whiteshoemedia/gondola-core/archive/master.zip unzip master.zip {% endhighlight %}

Install dependencies

For most Linux-based systems the setup.py script will install all Python dependencies. {% highlight 'bash' %} cd liftpass python setup.py {% endhighlight %}

Configuring Liftpass

Liftpass is configured via the config.py file. This file should have been configured for you while running the setup.py script, but if it didn't you can always re-run:

{% highlight 'bash' %} python setup.py config {% endhighlight %}
By default Liftpass is configured to use SQLite for all database needs. This is a great solution to evaluate Liftpass in a development environment, but it is not intended for production settings. Liftpass will work with most SQL based databases that SQLAlchemy supports.

Starting the server

You are now ready to manually start running ths server.

{% highlight 'bash' %} python manage.py API start {% endhighlight %} {% endblock %}
